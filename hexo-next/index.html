Hexo 主题 NexT 配置

发表于 2023-11-06 更新于 2024-08-10 阅读次数：
我认为还是有必要将 Hexo 的 NexT 主题相关内容单独作为一篇来记的。作为 Hexo 的经典主题，NexT 各方面都还是很完备的，其配置过程比较方便。我这篇的架构就是按官方文档来的，但是具体内容就不照搬了，以网址链接的形式给出，自己只是给些补充。

官方文档

https://theme-next.js.org/

NexT 主题的官网地址改过有几次，以后并不确定还会不会接着改（目前看来应该是不会了）。

Getting Started

https://theme-next.js.org/docs/getting-started/

https://theme-next.js.org/docs/getting-started/installation

https://theme-next.js.org/docs/getting-started/configuration

主要是关于安装，部署，配置文件的内容。

关于 GitHub 仓库克隆和 npm 安装，这两种安装方式的情况和具体实践，可以看我另一篇博客，关于 Hexo 搭建。那里有具体说明。

Theme Settings

https://theme-next.js.org/docs/theme-settings/

修改主题 Scheme，一共有四种主题可选。主题可以设置为深色模式 Dark Mode，将darkmode的值设置为true即可。

Sidebar

https://theme-next.js.org/docs/theme-settings/sidebar

侧边栏，其中可以设置头像 avatar 相关的内容。除了选择头像图片之外，还可以设置是否显示为圆形rounded，是否可旋转rotated，也就是光标移动到头像上时，头像是否会跟着旋转。

Posts

https://theme-next.js.org/docs/theme-settings/posts

NexT 主题下，主页的文章默认是内容全部显示的。与大部分主题类似，可以通过在文章中插入<!-- more -->使得只预览其前面的内容，或者在 markdown 文件上方Front-matter中，设置description的值，即为预览的内容。

Miscellaneous

https://theme-next.js.org/docs/theme-settings/miscellaneous

有关于代码块 CodeBlock 的设置，比如是否有复制按钮 copy_button，等等。

Third Party Plugins

https://theme-next.js.org/docs/third-party-services/

Math Equations

https://theme-next.js.org/docs/third-party-services/math-equations

关于 \(\LaTeX\) 公式渲染。注意，markdown 的行内公式要求，两侧的$符号要紧挨着中间的公式内容，不能有空格，而要与外面的文字之间有空格。

这里说到了公式渲染可能出现的问题，需要我们手动卸载hexo-renderer-marked，安装hexo-renderer-pandoc。

另外，电脑本地也要安装 pandoc，直接去官网安装。安装完最好重启电脑（不重启好像也行，反正要退出命令行重进）。

External Libraries

https://theme-next.js.org/docs/third-party-services/external-libraries

可以设置页面加载的动画效果。建议把motion里的async设置为true，这样，侧边栏，文章等内容能够一同加载，否则如果文章较多，需要文章一篇一篇加载完后再显示侧边栏。

Comment Systems

https://theme-next.js.org/docs/third-party-services/comments

这些评论系统大部分是基于 GitHub 的 issues，我试了其中的一些，比较推荐 Utterances。

而 Gitalk 在使用的安全性和配置的便利性来说不如 Utterances，而且 Utterances 还有邮件通知的功能。我看了一下原理，大概是你评论的时候，你就会订阅相应的 issues，如果 issues 有别人评论，机器人就会给你发邮件进行通知。

https://utteranc.es/

除了这些之外，还有 Giscus，他则是基于 GitHub 的 discussions，优点是能够对评论进行回复，缺点则是没有邮件通知，只是在一开始创建讨论的时候有通知，后面的就不行。但还是把相关资料贴在下面。

https://theme-next.js.org/plugins/

https://github.com/next-theme/hexo-next-giscus

https://giscus.app/zh-CN

Statistics and Analytics

https://theme-next.js.org/docs/third-party-services/statistics-and-analytics

关于数据分析，比如可以显示网站的访问人数和次数，等等。一般是使用不蒜子 Busuanzi 即可。

Search Services

https://theme-next.js.org/docs/third-party-services/search-services

这里只需要添加本地搜索功能即可。

Advanced Settings

https://theme-next.js.org/docs/advanced-settings/
