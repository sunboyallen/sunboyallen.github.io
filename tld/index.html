关于 TLD

发表于 2023-11-08 更新于 2024-12-05 阅读次数：
如题，基本是贴上官方网址，我自己就不多说了，只是给一些补充。

https://www.icann.org/

https://icannwiki.org/

ICANN wiki，可以查各个 TLD 的信息。

https://www.iso.org/iso-3166-country-codes.html

ISO 3166 标准。

https://www.iana.org/

https://www.iana.org/domains/root/db

https://data.iana.org/TLD/tlds-alpha-by-domain.txt

可以查看 root zone，查看根域名。

eTLD（effective TLD），也就是 public suffix，由 Mozilla 提倡。

https://developer.mozilla.org/en-US/docs/Glossary/eTLD

https://publicsuffix.org/

https://publicsuffix.org/list/public_suffix_list.dat

这也是 Python 的 tldextract 库用到的 public suffix 列表。

https://newgtlds.icann.org/en

https://ntldstats.com/

https://ntldstats.com/tld

上面的是各 nTLD 的数据信息。没事的话可以查查，有些很有意思的域名。

看看 xyz 的官网（笑）。
