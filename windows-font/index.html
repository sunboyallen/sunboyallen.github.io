关于微软 Windows 系统字体及误区

发表于 2024-01-06 更新于 2024-08-10 阅读次数：
如题。主要还是给出相关资料，并且自己做点补充。

写在开头

先给出微软官方的相关资料。

https://learn.microsoft.com/zh-cn/typography/font-list/

Windows 系统自带的字体介绍基本都在这里面了。

误区

估计是为了便于用户使用和选择，微软将一些字体名字在显示上做了修改简化。比如，在 Windows 最常见到和使用的宋体，黑体，等等，其实全名是中易宋体，中易黑体，分别叫 SimSun，SimHei。

https://learn.microsoft.com/zh-cn/typography/font-list/simsun

https://learn.microsoft.com/zh-cn/typography/font-list/simhei

楷体是 KaiTi，仿宋是 FangSong，等等，这些也都是中易公司的。

https://learn.microsoft.com/zh-cn/typography/font-list/kaiti

https://learn.microsoft.com/zh-cn/typography/font-list/fangsong

某种程度上，这样确实便于电脑使用者快速选择所需字体，但这也影响了我们很多人的认知，以为宋体（或者说明朝体），黑体（或者说哥特体）等等，就只是电脑里那个选项的字体。实际上，所谓的宋体，黑体，是指这类字体风格，不是指具体某个公司制作的字体。
