查看自己 Windows 电脑的产品密钥（备忘）

发表于 2024-06-21 更新于 2024-08-10 阅读次数：
如题，作为备忘，毕竟是看了网上资料的，简单记录一下。

是看了这个知乎回答：https://www.zhihu.com/question/382395980/answer/1103795774

一个是原始产品密钥。回答说是用管理员身份运行 cmd。不过好像正常使用 PowerShell 运行命令就行。

另一个是备份产品密钥，在注册表查看。HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SoftwareProtectionPlatform，BackupProductKeyDefault文件内容就是。可以敲下面的命令，也可手动翻注册表。
